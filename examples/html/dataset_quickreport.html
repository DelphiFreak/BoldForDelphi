DataSet & Quick Report
Launch Delphi-example | Launch CBuilder-example

This example shows how to use a BoldDataSet together with report generation with the Quick report components in Delphi. The example model contains a Person - Building relationship where a Person can have zero to one home and a Building can have zero to many residents.

How to run

Create instances of buildings and persons. Drag instances of persons to the residents listbox for a building. Then generate the report. The report will show a list of all the buildings and for each building its residents.

Use of BoldDataSet

This component builds a dataset of bold objects and is useful when creating reports. The BoldDataSet is connected to a list of bold objects and the desired object fields to show in the dataset are described.

When creating the report you connect the report fields to desired datasets and datafields. The report tool then loops the dataset and the selected datafields are shown in the report in runtime.

See also

  • Programmers Reference ([BfDInstallDir]/help/Bfd.hlp) for explanation of the properties on BoldDataSet.

Note! Please make sure to use the "Create DB & Alias" button before attempting to open the system for the first time. Once the database and alias is established you only have to use the "Open system" button when running the example program.
