SQL Handle
Launch Delphi-example | Launch CBuilder-example

This example shows how to use a BoldSQLHandle.

How to run

The grid with persons is populated with objects. Set values to the Where and Order fields and click the button Execute SQL. 

Use of BoldSQLHandle

The BoldSQLHandle fetches objects from the database according to the properties set in the component. To display the result you must use a TBoldCursorhandle or TBoldListHandle. Observe that only objects that are stored persistent are fetched.

In this example the BoldSQLHandle fetches person objects from the database. When pressing the Execute SQL button the SQLWhereClause and SQLOrderByClause property in the BoldSQLHandle component are set. To run the query the ExecuteSQL method on the component is executed.

The need for a SQLhandle has been somewhat reduced after introducing the EvaluateInPS property on BoldListHandles. Setting the property to true transforms an OCL expression to a SQL expression. Review the online documentation for more information on this topic.

See also

  • Programmers Reference ([BfDInstallDir]/help/Bfd.hlp) for explanation of the properties on BoldSQLHandle.

Note! Please make sure to use the "Create DB & Alias" button before attempting to open the system for the first time. Once the database and alias is established you only have to use the "Open system" button when running the example program.
