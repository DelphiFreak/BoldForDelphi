Derived Handle
Launch Delphi-example | Launch CBuilder-example

This example shows how to use derived handles. Derived handles are used when you want to derive the object(s) the handle points to.

How to run

The upper grid shows persons and their assets and the lower grid shows those persons whose assets exceed the value set by the trackbar under the upper grid. You can also control the number of persons shown in the lower grid by changing the number in the edit field just above the grid.

Change the assets limit and the number of persons in the list limit to see what happens in the lower grid. Also change the names and assets in the grids to see how that reflects one another.

Use of BoldDerivedHandle

Implement the event OnDeriveAndSubscribe on the BoldDerivedHandle component to do the derivation and place appropriate subscriptions.

This example subscribes to the breakpoint for being rich, the number of richest persons (these two are held in variable handles), the M_Assets member of each person and the list of persons. If any of these change, the system has to derive again.

See also

  • Where to use subscriptions (also found in [BfDInstallDir]/help/Bfd.hlp)
  • Programmers Reference ([BfDInstallDir]/help/Bfd.hlp) for explanation of the properties on BoldDerivedHandle.
