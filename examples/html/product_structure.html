Product Structure
Launch Delphi-example | Launch CBuilder-example

This example shows how to create recursive structures and display them in a TBoldTreeView.

Note that though this example contains very little code, several pieces of the functionality could be handled automatically if added to the model. The rationale for not implementing the example that way is that we want the examplify some code...
(Hint: derived attribute)

How to run

When you open the system some products are created automatically. You can see how that is done in the BoldActivateSystemAction1SystemOpened method in the main form. 

The products are either simple or assembly products. An assembly product consists of other products. You can get a recursive structure by adding an assembly product as a part of another assembly product.
