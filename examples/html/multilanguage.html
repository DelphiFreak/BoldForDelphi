MultiLanguage
Launch Delphi-example

This example requires the MLTestAttributesD5 or MLTestAttributesD6 packages to be installed. (D5 or D6 corresponds to your Delphi version.)

$(PRODUCT_BASE_NAME) includes the possibility to store and retrieve data in several concurrent languages. This example illustrates how this is made possible by the use of a few specialised controls.

In a multilang application there are a number of constraints:

  • There must be a class in the model that represents the languages. The name of this class must correspond to the variable BoldMLLanguageClassName (default 'LanguageClass').
  • The language class must have two attributes corresponding to:

    BoldMLLanguageNameAttributeName: String = 'LanguageName';
    BoldMLLanguageNumberAttributeName: String = 'LanguageNumber';

  • The language class can not have any multilingual attributes and the LanguageNumbers and LanguageNames must be unique!

Before any objects with Multilingual attributes are created, there must be at least one instance of the language class.

The set of languages (and the legal values of multilingual valuesets) are initialized when they are first accessed. Adding a new language in runtime will not have any effect until the next time the program is started.

The multilingual attributes will depend on the primary and secondary language of the application. (These are of the type TBALanguage). Changing the primary or secondary language will immediately change the string representation of the multilingual attributes. If an attribute does not have a value for the primary language, the value for the secondary language will be shown in angle brackets (<Swedish: xxx >). if there is no value for the secondary language either, the attribute will use the first value it can find and display it in double angle brackets (<< Danish: yyy >>).

When editing a multilingual string, it is normally the primary language that is edited, even if the secondary language is displayed.

Using a TBoldMLRenderer, you can force the GUI to always show and edit the value of a specific language representation.
