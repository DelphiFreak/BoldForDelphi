BoldImage
Launch Delphi-example | Launch CBuilder-example

The BoldImage control is used to view images of different formats. This example illustrates the some of the settings and functions in the control. The features included in the example are:

  • Loading from and saving to files.
  • Using the Windows Clipboard.
  • Stretching and scaling the image.

How to run

In this example application you can load and save image files. All the images you load or paste into the application are added to a list. The navigator at the bottom of the form is used to navigate that list. Click the add-sign on the navigator to reserve space in the list were you can insert a new image. To insert a new image you can either load it from a file on your harddisk or you can copy or cut the image from a previous space in the list and paste it on the new space.

To scale the image use the button View.

See also

  • Programmers Reference ([BfDInstallDir]/help/Bfd.hlp) for explanation of the properties on BoldImage.

Note! Please make sure to use the "Create DB & Alias" button before attempting to open the system for the first time. Once the database and alias is established you only have to use the "Open system" button when running the example program.
